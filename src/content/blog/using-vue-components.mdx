# Using Vue Components in Blog Posts

This is a blog post that uses Vue components directly in the content. MDX allows you to import and use components right in your markdown.

<PageSection backgroundColor="yellow" width="800">

</PageSection>

## Why use MDX?

MDX combines the power of markdown with the flexibility of JSX. This means you can:

1. Write content in markdown syntax
2. Import and use components
3. Use JavaScript expressions
4. Create dynamic content

## Example with dynamic content

Here's an example of using JavaScript in your MDX:

export const favoriteFrameworks = ['Vue', 'React', 'Svelte', 'Angular'];

<ul>
  {favoriteFrameworks.map(framework => (
    <li key={framework}>{framework}</li>
  ))}
</ul>

Try it out in your own blog posts!
